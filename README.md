# Интерактивный справочник электросхем Volvo XC70

Веб-справочник для Volvo XC70 3.2 2008 года: поиск по номеру и цвету провода, разъёму, компоненту или модулю с обязательной ссылкой на страницу исходной схемы.

## Важно

Исходный файл — руководство 2014 года для XC70 (08-). Оно является справочным источником, а не подтверждением конфигурации конкретной машины 2008 года. Перед подключением всегда сверяйте VIN, код двигателя, разъём и реальную проводку. Не вмешивайтесь в SRS/SIPS и силовые цепи без профильной документации и безопасного обесточивания автомобиля.

## Локальная обкатка (без деплоя на VPS)

Прод на `195.19.12.173` трогайте только после проверки здесь.

```powershell
cd C:\Users\eni19\volvo-xc70-wiring
npm install
npm run dev
```

Или одной командой: `npm run dev:local` (создаст `.env` из примера, если его нет).

| Что | URL |
|-----|-----|
| UI (Vite) | http://localhost:5173 |
| API | http://localhost:3000 (`/api` проксируется с UI) |

Нужны локально (уже в репо / на диске):
- `data/wiring.sqlite`, `data/dtc.sqlite`
- `data/ewd/*_index.json` и дерево SVG `data/ewd/ewd_source/…`

Правки в `client/` и `server/` подхватываются сразу (HMR / tsx watch).  
На VPS деплойте только когда локально ок: `git push` → на сервере `bash deploy.sh` (без `BUILD=1`, если образ уже есть).

PDF-мануал для основного сценария EWD/VIDA не обязателен; импорт PDF — опционально:

```powershell
npm run import:manual -- "C:\path\to\Электросхемы XC70.pdf"
```

## Данные и исправления

- `scripts/import-wiring-manual.ts` извлекает текст PDF, номера страниц и распознаваемые обозначения проводов/цветов/разъёмов.
- `data/overrides/xc70-2008.json` содержит безопасные ручные алиасы и заметки, специфичные для XC70 3.2 2008. Все такие дополнения должны иметь источник.
- Для резервной копии сохраните `data/wiring.sqlite` вместе с файлом overrides.
